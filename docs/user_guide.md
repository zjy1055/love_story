## 概述

恋爱故事记录应用是一款本地运行的桌面应用，专为记录和管理恋爱回忆而设计。通过这款应用，您可以：

- 记录重要的恋爱事件和日期
- 上传和管理珍贵的照片
- 设置纪念日倒计时
- 添加个人格言和相处守则
- 定期备份数据以确保安全

## 首次使用

### 启动应用

1. **Windows 用户**：
   - 双击 `恋爱故事记录应用.exe` 文件启动
   - 首次启动时，应用会在用户目录下创建必要的数据文件夹

2. **开发者运行**：
   - 确保已安装 Python 3.6+ 和所有依赖
   - 在命令行中运行 `python main.py`

### 数据存储

应用的数据默认存储在以下位置：
- Windows: `C:\Users\[您的用户名]\.love_story_app\`
- 包含：数据库文件、上传的照片和备份文件

## 首页功能

首页是应用的主要入口，展示了以下内容：

### 照片轮播

- 自动轮播您上传的精选照片
- 每张照片显示3秒后自动切换到下一张
- 点击照片可以进入照片查看器查看大图

### 纪念日倒计时

- 显示距离下一个重要纪念日的天数、小时数、分钟数和秒数
- 纪念日信息可以在配置页面中设置

### 个人格言和相处守则

- 显示您添加的个人格言和恋爱相处守则
- 这些内容可以在配置页面中编辑和更新

## 事件管理

事件管理功能允许您记录和查看恋爱中的重要时刻。

### 添加新事件

1. 点击导航栏中的"事件记录"
2. 点击"添加事件"按钮
3. 在弹出的表单中填写：
   - 事件标题
   - 事件日期
   - 事件描述
4. 点击"保存"按钮完成添加

### 编辑事件

1. 在事件列表中找到要编辑的事件
2. 点击该事件对应的"编辑"按钮
3. 修改事件信息
4. 点击"保存"按钮更新

### 删除事件

1. 在事件列表中找到要删除的事件
2. 点击该事件对应的"删除"按钮
3. 在确认对话框中点击"确定"删除

### 搜索和筛选事件

- 使用顶部的搜索框可以按标题或描述搜索事件
- 搜索会实时更新结果，无需点击搜索按钮

## 照片管理

照片管理是应用的核心功能之一，提供了丰富的照片上传、查看和管理选项。

### 单张照片上传

1. 点击导航栏中的"照片管理"
2. 点击"上传照片"按钮
3. 在上传对话框中：
   - 选择照片文件（支持常见图片格式如 JPG、PNG 等）
   - 填写照片描述
   - 选择关联的相册（可选）
   - 选择关联的事件（可选）
   - 添加标签（多个标签用逗号分隔）
4. 点击"上传"按钮

### 批量照片上传

1. 点击导航栏中的"照片管理"
2. 点击"批量上传"按钮
3. 在批量上传对话框中：
   - 选择多张照片文件
   - 设置统一的相册（可选）
   - 设置统一的事件（可选）
   - 添加统一的标签（可选）
4. 点击"开始上传"按钮

### 照片查看

- **缩略图查看**：照片管理页面以网格形式显示所有照片的缩略图
- **大图查看**：点击任意缩略图进入大图查看模式
  - 使用左右箭头键或点击屏幕左右区域切换照片
  - 点击"X"按钮或按 ESC 键关闭查看器

### 照片搜索

#### 快速搜索

1. 在照片管理页面顶部的搜索框中输入关键词
2. 系统会自动搜索照片描述和标签中包含该关键词的照片

#### 高级搜索

1. 点击"高级搜索"按钮
2. 在高级搜索对话框中设置筛选条件：
   - 关键词：搜索描述和标签
   - 日期范围：筛选特定时间拍摄的照片
   - 相册：选择特定相册
   - 事件：选择特定事件
   - 标签：选择特定标签
3. 点击"搜索"按钮查看结果
4. 点击"重置"按钮清除所有筛选条件

## 配置设置

在配置页面中，您可以自定义应用的各项设置。

### 基本信息设置

1. 点击导航栏中的"配置"
2. 填写以下信息：
   - 纪念日日期：设置重要的纪念日
   - 纪念日名称：为纪念日添加名称
   - 个人格言：添加一句表达感情的格言
   - 相处守则：添加恋爱相处的规则或承诺
3. 点击"保存设置"按钮

### 相册管理

在配置页面的相册管理区域，您可以：

1. 点击"添加相册"创建新的照片分类
2. 为新相册输入名称
3. 保存后，该相册将在照片上传时可选

## 数据备份与恢复

为了确保数据安全，应用提供了完整的数据备份与恢复功能。

### 创建数据备份

1. 在配置页面中，找到"数据备份与恢复"区域
2. 点击"创建备份"按钮
3. 系统将自动创建当前所有数据的备份
4. 备份文件将保存在用户目录下的`.love_story_app/backups/`文件夹中
5. 备份成功后，备份列表会自动更新

### 查看备份列表

1. 在配置页面中，滚动到"备份列表"区域
2. 您可以看到所有已创建的备份，包括：
   - 备份时间
   - 文件大小
   - 备份状态

### 从备份恢复

#### 方法一：从备份列表恢复

1. 在备份列表中找到要恢复的备份
2. 点击该备份对应的"恢复"按钮
3. 在确认对话框中点击"确定"
4. 系统将从该备份恢复数据
5. 恢复完成后，页面将自动刷新

#### 方法二：上传备份文件恢复

1. 点击"从文件恢复"按钮
2. 选择您保存的备份文件（.zip格式）
3. 点击"打开"开始恢复
4. 恢复完成后，页面将自动刷新

### 清理旧备份

为了节省空间，您可以清理较早的备份：

1. 点击"清理旧备份"按钮
2. 系统将自动删除超过保留期的备份文件
3. 清理完成后，将显示删除的备份数量

## 常见问题解答

### 1. 应用程序无法启动怎么办？

- 确保您的操作系统满足要求（Windows 7以上或兼容系统）
- 检查是否有其他实例正在运行
- 尝试以管理员身份运行
- 检查是否有权限创建文件和文件夹

### 2. 照片上传失败怎么办？

- 检查照片文件格式是否支持（JPG、PNG等）
- 确保文件大小不超过系统限制
- 检查存储空间是否充足

### 3. 如何在不同设备间迁移我的数据？

- 在源设备上创建备份
- 将备份文件复制到目标设备
- 在目标设备上使用"从文件恢复"功能

### 4. 我可以自定义照片存储位置吗？

目前不支持自定义存储位置，所有数据统一存储在用户目录下的`.love_story_app/`文件夹中。

### 5. 应用程序支持哪些语言？

当前版本仅支持中文界面。
